# Marketing Content

This folder contains marketing and promotional content for Speedkit.

## Structure

```
marketing/
├── README.md           # This file
└── linkedin/           # LinkedIn posts organized by feature
    ├── README.md       # Content strategy & templates
    ├── speedkit/       # Overall product posts
    ├── critical-css/   # Critical CSS feature posts
    └── preload-tags/   # Preload feature posts
```

## Content Guidelines

### Voice & Tone

- **Developer-first** — Write for engineers, not marketers
- **Tinkerer mindset** — Share experiments and learnings
- **Honest** — Acknowledge limitations and works-in-progress
- **Technical depth** — Include code snippets and real numbers

### Post Types

1. **Introduction** — Feature launch, what it does
2. **Deep Dive** — Technical implementation details
3. **Results** — Case studies with real metrics
4. **Update** — Improvements and iterations

### Hashtags

```
Performance: #WebPerformance #CoreWebVitals #PageSpeed #LCP
Technical:   #JavaScript #CSS #WebDev #FrontendDevelopment
Community:   #OpenSource #BuildInPublic #DevTools
```

## Workflow

1. Draft post in appropriate feature folder
2. Update status: Draft → Ready → Posted
3. Add posting date and URL when published
4. Track engagement metrics

See [linkedin/README.md](linkedin/README.md) for detailed content strategy.
